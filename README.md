# イベント告知サイト テンプレート

チームで作る架空イベントの告知サイトです。
Git グループワーク研修の演習用テンプレートです。

---

## ページ構成

1ページ構成（`index.html`）でスクロール型のサイトです。

| セクション | 担当 | ブランチ |
|-----------|------|---------|
| ヒーロー・ヘッダー・フッター | A | `feature/issue-1` |
| 概要（日時・場所・料金・定員） | B | `feature/issue-2` |
| プログラム | C | `feature/issue-3` |
| アクセス + JavaScript | D | `feature/issue-4` |

---

## ファイル構成

```
repo-template/
├── index.html   ... メインページ（全セクション）
├── style.css    ... 全員が編集する共通スタイル ★コンフリクット対象
├── script.js    ... JavaScript
└── README.md    ... このファイル
```

---

## 開発ルール

- `main` ブランチには直接 push しない
- 作業は必ず `feature/issue-番号` ブランチで行う
- マージ前にチームメンバーにレビューを依頼する
- コミットメッセージは `feat:` `fix:` `style:` などの種別をつける

---

## 使い方

`index.html` をブラウザで開いてください。
サーバー不要でローカル環境のみで動作します。
